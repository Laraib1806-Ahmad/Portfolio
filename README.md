# Laraib Ahmad - MERN Stack Developer Portfolio

A modern, responsive portfolio website showcasing my skills, projects, and experience as a MERN Stack Developer.

## 🚀 Features

- **Dark Theme**: Professional dark mode with sky blue accent color (#38bdf8)
- **Smooth Animations**: Framer Motion powered scroll-based animations
- **Responsive Design**: Fully responsive layout that works on all devices
- **Modern UI**: Clean and professional design with subtle hover effects
- **Sticky Navigation**: Header with fade-in animation and backdrop blur on scroll
- **Smooth Scrolling**: Seamless navigation between sections
- **Project Showcase**: Animated project cards with hover effects and video demo support
- **Interactive Skills**: Stagger animations and hover lift effects
- **Timeline Experience**: Animated timeline for work experience and education
- **Contact Section**: Animated contact cards with hover effects
- **GitHub Integration**: Direct links to GitHub repositories

## 🛠️ Tech Stack

- **React** - Frontend framework
- **Vite** - Build tool and dev server
- **Tailwind CSS** - Styling with custom dark theme
- **Framer Motion** - Smooth animations and transitions
- **Lucide React** - Modern icon library

## 📦 Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd portfolioooo
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Build for production:
```bash
npm run build
```

## 🎨 Sections

1. **Header**: Sticky navigation with fade-in animation, backdrop blur, and underline hover effects
2. **Hero**: Animated introduction with gradient background, floating elements, and staggered text animations
3. **About**: Scroll-triggered fade-in animations with slide-from-left text
4. **Skills**: Staggered card animations with hover lift effects and glowing borders
5. **Projects**: Scale and fade animations with hover effects, pulsing video buttons
6. **Experience**: Timeline-style layout with animated dots and alternating slide animations
7. **Contact**: Fade-up animations with interactive hover effects on contact cards
8. **Footer**: Simple fade-in with animated heart icon

## 📝 Customization

To customize this portfolio:

1. Update personal information in each component
2. Add your project details in `src/components/Projects.jsx`
3. Update skills in `src/components/Skills.jsx`
4. Modify colors in `tailwind.config.js`
5. Add your resume PDF to the public folder and update the link in Hero component
